+A Climate Model for Earth & Mars

+ +
    +
  • Free - All components are open source
    • +
  • Interactive - Graphical two-way user interface
    • +
  • Modular - Problem oriented model configuration
    • +
  • Parallel - Runs on parallel machines and clusters
    • +
  • Scalable - Wide range of resolutions starting at T21 L5
    • +
  • Portable - Hardware independent code
    • +
  • Structured - Clear and rich commented source code (Fortran 90)
    • +
  • Compatible - supports MPI, GRIB, NetCDF
    • +
  • Documented - Extensive Users's Guide and Reference Manual
    • +
  • Fast - Simulate 500 years per day on a 4 core CPU
    • +
  • Postprocessing -The "burner" postprocessor
